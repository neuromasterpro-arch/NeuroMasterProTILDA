У меня есть сайт на платформе Tilda. Это сайт по регенерации текста, по сканированию картинок. https://neuromasterpro.ru/ Я хочу делать сервис полезный, интересный для людей, для работы с искусственным интеллектом. Работаю я с ChatGPT, OpenAI по API. В файлах написан код для каждого отдельного блока T123 на тильде. Весь код разбит на отдельные блоки для того, чтобы я мог работать с этим кодом как с конструктором. То есть там, на тех страницах, где нужен голосовой ввод, я ставлю блок для голосового ввода. На тех страницах, где надо делать озвучку текста, я ставлю этот блок. Таким образом, у меня есть много разных блоков для создания нужного функционала для данной страницы. Это сделано для того, чтобы там, где нужен определенный функционал, чтобы не нагружать страницу сайта лишним кодом.Я стремлюсь сделать сайт SEO-оптимизированным для того, чтобы поисковые системы могли высоко его оценить и показывать на первой странице всех поисковых систем.
