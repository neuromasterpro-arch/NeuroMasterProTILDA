## Как внести и зафиксировать свои изменения
1. Отредактируйте нужные файлы в рабочей копии.
2. Добавьте их в индекс командой `git add <имя_файла>`.
3. Зафиксируйте правки: `git commit -m "Ваше описание"`.
4. Отправьте результат на сервер через `git push`, после чего другие пользователи смогут получить ваши изменения через `git pull`.
